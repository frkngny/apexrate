# ApexRate

This project is created to learn the concept behind a Dynamic Real-Time Inventory & Surge Pricing Engine.

I will create distributed systems with SAGA pattern, idempotency and distributed lock. Event-driven architecture with eventual consistency.

## Tech Stack
- Language: Java - JDK 21
- Framework: SpringBoot 3.x
- DB: 
    - PostgreSQL for pricing.
    - Redis for stock and momentary price caching.
- Event-Driven: Apache Kafka
